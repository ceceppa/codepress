Create your own generator

codepress.io is based on WordPress, so to start:

  • download and configure WordPress on your machine.
  • download and install the WP REST API plugin.
  • download, install and active the codepress.io theme
we're almost done, you choose between the following boilerplates:
  • dynamic form - contains the basic code to generate dynamic forms.
  • basic - contain the basic code to make the addon work, you can customise the full interface and routes.
For further visit the boilerplates github page.

Documentation

Don't forget to have a look to the documentation.

Publish your own generator

Do you want to publish your generator on codepress.io?
Cool, just fill up the form and wait for the approval.

Your generator have to he hosted on github, otherwise the request will be automatically denied.
